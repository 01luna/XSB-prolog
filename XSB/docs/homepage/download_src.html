Download the Source Code

You can obtain the source for the XSB system by performing an anonymous ftp to ftp.cs.sunysb.edu. All XSB related files reside in the /pub/XSB directory and its subdirectories.

You can also download the zipped file of the XSB system by clicking one of the following links:

  • XSB.tar.gz
    This file includes all source code, examples, and manual.ps.

  • XSB.tar.Z
    This file includes all source code, examples, and manual.ps)

  • XSB95.zip
    This is a WinZip file for Windows 95/NT, which includes source code, examples, manual.ps, xsb.exe (without ODBC interface), and xsb_odbc.exe (with ODBC interface).
