The XSB Research Group

Our main interests are Logic Programming, Deductive Databases and Non-Monotonic reasoning. Currently we have projects on: different (more set-at-a time) scheduling strategies for tabled evaluations, parallelization models for XSB, implementation of full SLG, recursive query optimization...

We have developed XSB a research-oriented Logic Programming and Deductive Database system for Unix and DOS-based systems.
